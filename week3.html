🤖 Add an AI-Powered Function to Your App

Integrate an AI-powered feature using Together.ai (or similar) to your app! 🚀

Adding AI capabilities can make your app more interesting if there are usecases related to summarizing reports, generating content, or providing insights. 🧠

You can use this hospital patient management project as a reference: Github Repo

📝 Action Items:

  1. 🔑 Sign up for Together.ai (or similar AI service like OpenAI, Anthropic) and get your API key
  2. 🔒 Secure your API key:
    • Create a .env file in your project root
    • Add your API key there: TOGETHER_API_KEY=your_key_here
    • Add .env to your .gitignore file to prevent accidental leakage (it will prevent github from pushing it to the repo)
    • For a quick test, copy the main.py file here Github Repo and run it to see if you can get a response from the AI model
    . You can use the prompt_llm function to create your AI function for your app.
  3. 🛠️ Implement an AI feature:
    • If your app naturally uses AI:
      • Add any feature like report summarization, content generation, or data analysis
    • If your app doesn't naturally use AI:
      • Integrate a chatbot where users can learn about your app's services
  4. 📸 Share a screenshot of your AI-powered function output in the 🤖｜week-3-outputs Discord channel

🚀 Deploy to Hugging Face Spaces

Deploy your AI app to Hugging Face Spaces to make it publicly accessible and showcase your work! 🌐

Hugging Face Spaces provides free hosting for AI applications, making it perfect for sharing your project with the world. 🌟

📝 Action Items:

  1. 🌐 Go to Hugging Face Spaces and click "New Space"
  2. 📝 Configure your Space:
    • Add a name for your space
    • Select Docker option under Space SDK
    • Make sure it's set to Public
  3. 📁 Upload necessary files by going to "Files":
    • Dockerfile
    • requirements.txt
    • app.py and related app files

    • you can use this Omniscient Hugging Face Space as reference: HF Space.
  4. 🔐 IMPORTANT - Secure your API key:
    • DO NOT upload your .env file with the API key
    • Go to Settings → Variables and Secrets
    • Add your API key as a secret there so no one can see it
  5. ✅ Verify deployment:
    • Check if your space is in "Running" mode
    • Test that your app works correctly
  6. 📤 Share your Hugging Face Spaces URL in the 🤖｜week-3-outputs Discord channel
