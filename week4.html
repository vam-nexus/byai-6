🧠 Add RAG to your App

RAG allows your AI to access and cite specific documents or knowledge bases, making responses more accurate and trustworthy. 📚

You can use this RAG code for inspiration: Github Repo

  • Run python rag_cmd.py to see the RAG code in action

📝 Choose Your RAG Implementation (Watch the Youtube Video for support if needed):

  1. 🤖 Option 1: Enhance Your Chatbot with RAG:
    • Make your chatbot access documentation (README files, websites)
    • Allow it to answer questions based on specific knowledge sources
    • Include citations showing where information came from
  2. ⚙️ Option 2: Add RAG to a Specific Function:
    • Use RAG for domain-specific tasks (e.g., legal documents, technical manuals)
    • Load .txt files or documents about your topic/domain
    • Have the AI model use this knowledge to provide specialized answers
  3. 📊 Implementation Requirements:
    • Output must include citations of information sources
    • Show users where the AI got its information from
    • Test with different types of queries to ensure accuracy
  4. 📤 Share your RAG-powered output (with citations!) in the 🚀｜week-4-outputs Discord channel

🎯 RAG Implementation Ideas:

  • Customer support bot that references product documentation
  • Research assistant that cites academic papers or articles
  • Code helper that references GitHub repos or technical docs
  • Domain expert (legal, medical, financial) with specialized knowledge
