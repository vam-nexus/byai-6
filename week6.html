🌐 (Optional) Host your App on a Custom Domain

⚠️ This is an OPTIONAL task - you do not have to complete this week!

🧱 1. Create your Flask project

Make a new folder: mkdir flask-app && cd flask-app

Create app.py and requirements.txt with Flask and gunicorn

🧑‍💻 2. Push to GitHub

git init → git add . → git commit -m "initial commit" → git push

🚀 3. Deploy on Render

Go to render.com → Sign up with GitHub → New Web Service → Choose your repo

Settings: Python, pip install -r requirements.txt, gunicorn app:app, Free plan

🌐 4. Add Custom Domain

Note: Cloudflare requires you to pay for a custom domain. Cloudflare is one of the best for this.

Render → Settings → Custom Domains → Add your domain

Get DNS target from Render

☁️ 5. Configure Cloudflare DNS

Add CNAME record: www → your-app-name.onrender.com

🔒 6. SSL/HTTPS

Cloudflare: SSL mode to Full (Strict) | Render: Enable Force HTTPS

Alternatives: You can use Hugging Face Spaces instead, but Render.com might work better. There are other hosting providers too.

🔁 Auto-updates: Push to GitHub → Render auto-redeploys! 🎉

💳 (Optional) Integrate Stripe Payments with your App

⚠️ This is an OPTIONAL advanced task!

💻 1. Install & Configure

Add to requirements.txt: stripe, python-dotenv

Get API keys from Stripe → Developers → API Keys:

  • STRIPE_PUBLISHABLE_KEY = pk_test_... (Publishable key)
  • STRIPE_SECRET_KEY = sk_test_... (Secret key)
  • PRICE_ID = price_... (Create Product → Price, copy Price ID)

Set these in Render → Service → Environment

🐍 2. Flask Backend

Add routes: /create-checkout-session, /success, /webhook

Use stripe.checkout.Session.create() for payments

🌐 3. Frontend

Create templates/ with index.html, success.html, cancel.html

Use Stripe.js: stripe.redirectToCheckout()

⚙️ 4. Stripe Dashboard

Go to Stripe Dashboard → Create Product + Price → Copy Price ID

Add webhook endpoint: https://yourdomain.com/webhook

Use test mode with test cards (4242 4242 4242 4242)

💡 Test locally with Stripe CLI: stripe listen --forward-to localhost:10000/webhook
